## 版本历史

### v0.1.0 (2026-03-19)

#### 新增功能

- **核心终端功能**
  - 交互式终端 Shell
  - 命令执行引擎
  - 管道命令支持
  - 后台进程执行
  - 超时控制

- **安全管控系统**
  - 命令黑白名单（支持正则表达式）
  - 工具黑白名单（MCP 工具管控）
  - API 黑白名单（外部接口管控）
  - 禁止模式检测（密码、密钥等敏感信息）
  - 命令长度限制

- **运行模式**
  - **普通模式 (Normal)**: 允许执行大部分命令，仅检查黑名单
  - **严格模式 (Strict)**: 只允许执行白名单内的命令、工具和 API

- **操作行为监控**
  - 完整的活动日志记录
  - 命令执行审计
  - 安全违规检测
  - 日志轮转和导出
  - 统计分析功能

- **MCP 工具系统**
  - 文件系统工具（读写、列出目录）
  - Shell 执行工具
  - Web 搜索工具
  - 工具注册和管理框架

- **API 客户端**
  - HTTP 请求封装
  - GitHub API 支持
  - OpenAI API 支持
  - 自定义端点配置

- **Python 绑定**
  - Python API 封装
  - 安全策略模块
  - 监控模块
  - 交互式 Shell

#### 技术栈

- **Rust**: 核心引擎、性能关键模块
- **Python**: 高级 API、用户接口
- **PyO3**: Rust-Python 绑定
- **Tokio**: 异步运行时

#### 配置文件

- 支持 JSON 格式配置
- 默认配置路径: `~/.luoli/config.json`
- 日志路径: `~/.luoli/logs/activity.log`

## 兼容性

### 系统要求

- **操作系统**: Linux, macOS, Windows (WSL2)
- **Rust**: 1.70+
- **Python**: 3.8+
- **内存**: 最低 512MB，推荐 1GB+

### 依赖项

#### Rust 依赖
- tokio 1.35+
- serde 1.0+
- pyo3 0.20+
- clap 4.4+
- regex 1.10+
- reqwest 0.11+

#### Python 依赖
- 纯标准库实现
- 可选: pytest (测试), black (格式化)

## 更新日志格式

版本更新遵循 [Keep a Changelog](https://keepachangelog.com/) 规范。

### 变更类型

- `Added`: 新增功能
- `Changed`: 功能变更
- `Deprecated`: 即将移除的功能
- `Removed`: 移除的功能
- `Fixed`: 问题修复
- `Security`: 安全相关修复

## 版本号说明

采用 [语义化版本](https://semver.org/) 规范：**MAJOR.MINOR.PATCH**

- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能添加
- **PATCH**: 向后兼容的问题修复
