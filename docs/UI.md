# UI 界面设计文档

## 洛璃 (Luoli) - 用户界面设计

---

## 概述

洛璃终端助手提供三种用户界面：

1. **CLI (命令行界面)** - 传统终端交互
2. **TUI (终端用户界面)** - 图形化终端界面
3. **Web UI (网页界面)** - 基于浏览器的现代化界面

---

## 1. CLI 界面

### 特点
- 简洁的命令行交互
- 适合脚本和自动化
- 低资源占用

### 使用方式

```bash
# 基本命令行界面
luoli shell

# 执行单个命令
luoli exec "ls -la"

# 查看日志
luoli logs --limit 50

# 配置黑白名单
luoli config allow-command "^ls\\s*"
```

---

## 2. TUI 界面

### 特点
- 基于 `ratatui` 的高性能终端图形界面
- 支持键盘快捷键
- 实时显示日志和状态
- 无需浏览器

### 界面布局

```
┌─────────────────────────────────────────────────────────────┐
│                    🦞 洛璃终端助手                          │
├──────────────────────────────────────────┬──────────────────┤
│                                          │  [日志][安全][统计]│
│  终端输出区域                             │                  │
│                                          │  ✓ 10:30:45 ls   │
│  🦞 洛璃终端助手 v0.1.0                  │  ✗ 10:28:12 rm   │
│  当前模式: 普通模式                       │  ⊘ 10:25:33 curl │
│  输入 'help' 查看帮助                    │                  │
│                                          │                  │
│  luoli> help                             │                  │
│  可用命令:                               │                  │
│    help  - 显示帮助                      │                  │
│    mode  - 查看当前模式                  │                  │
│    clear - 清屏                          │                  │
│    exit  - 退出                          │                  │
│                                          │                  │
├──────────────────────────────────────────┴──────────────────┤
│  luoli> _                                                   │
├─────────────────────────────────────────────────────────────┤
│  模式: 普通模式 | 按 'm' 切换 | 按 'q' 退出 | Tab 切换面板   │
└─────────────────────────────────────────────────────────────┘
```

### 快捷键

| 按键 | 功能 |
|------|------|
| `q` | 退出程序 |
| `Ctrl+C` | 强制退出 |
| `Tab` | 切换到下一个面板 |
| `Shift+Tab` | 切换到上一个面板 |
| `1-5` | 快速切换标签页 |
| `m` | 切换运行模式 |
| `n` | 新建终端 |
| `Enter` | 执行命令 |
| `↑/↓` | 滚动历史记录 |

### 启动方式

```bash
# 启动 TUI 界面
luoli tui

# 以严格模式启动
luoli tui --mode strict
```

## 3. Web UI 界面

### 特点
- 现代化的网页界面
- 响应式设计，支持移动端
- 实时更新（WebSocket）
- 可视化图表和统计

### 界面组件

#### 3.1 侧边栏

```
┌──────────────┐
│    🦞 洛璃    │
├──────────────┤
│ ⌨️ 终端      │
│ 📋 日志      │
│ 🔒 安全      │
│ 📊 统计      │
│ ⚙️ 设置      │
├──────────────┤
│ 🛡️ 普通模式  │
└──────────────┘
```

#### 3.2 终端区域

- **标签页**: 支持多终端会话
- **终端输出**: 彩色代码高亮
- **命令输入**: 自动补全和历史记录
- **工具栏**: 清屏、复制、全屏

#### 3.3 右侧面板

**日志面板**:
- 实时命令执行记录
- 状态指示（成功/失败/阻止）
- 时间戳和命令详情

**安全面板**:
- 黑白名单管理
- 规则添加/删除
- 实时状态显示

**统计面板**:
- 执行统计卡片
- 图表可视化
- 趋势分析

### 主题配色

#### 深色主题（默认）

```css
:root {
  --bg-primary: #1a1a2e;      /* 主背景 */
  --bg-secondary: #16213e;     /* 次背景 */
  --bg-tertiary: #0f3460;      /* 第三背景 */
  --accent-color: #e94560;     /* 强调色 */
  --text-primary: #eaeaea;     /* 主文字 */
  --text-secondary: #a0a0a0;   /* 次文字 */
  --success-color: #4ecca3;    /* 成功 */
  --warning-color: #f9a825;    /* 警告 */
  --danger-color: #e94560;     /* 危险 */
}
```

### 启动方式

```bash
# 启动 Web 界面（默认端口 8080）
luoli web

# 指定端口
luoli web --port 3000
```

### 访问地址

启动后通过浏览器访问：
- http://localhost:8080
- http://127.0.0.1:8080

## 5. 界面设计原则

### 5.1 一致性
- 所有界面使用相同的配色方案
- 统一的图标和符号
- 一致的交互模式

### 5.2 可访问性
- 支持键盘导航
- 清晰的视觉反馈
- 适当的颜色对比度

### 5.3 响应性
- 快速响应用户操作
- 实时更新状态
- 流畅的动画效果

### 5.4 安全性可视化
- 严格模式使用红色标识
- 普通模式使用绿色标识
- 安全违规使用警告色

## 7. 开发指南

### 添加新的 TUI 组件

```rust
// 在 tui.rs 中添加
fn render_custom_widget(f: &mut Frame, app: &TuiApp, area: Rect) {
    let widget = Paragraph::new("自定义内容")
        .block(Block::default().title("标题").borders(Borders::ALL));
    f.render_widget(widget, area);
}
```

### 添加新的 Web 页面

```html
<!-- 在 web/ 目录下创建新页面 -->
<!DOCTYPE html>
<html>
<head>
    <title>新页面 - 洛璃</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <!-- 页面内容 -->
    <script src="app.js"></script>
</body>
</html>
```

## 8. 性能优化

### TUI 优化
- 使用双缓冲减少闪烁
- 限制刷新率（250ms）
- 按需渲染组件

### Web UI 优化
- 虚拟滚动处理大量日志
- WebSocket 增量更新
- 懒加载非关键组件

## 9. 故障排除

### TUI 显示问题

**问题**: 界面显示乱码

**解决**:
```bash
# 检查终端支持
export TERM=xterm-256color

# 或使用支持的终端
luoli tui
```

### Web UI 连接问题

**问题**: 无法访问 Web 界面

**解决**:
```bash
# 检查端口占用
lsof -i :8080

# 使用其他端口
luoli web --port 3000
```
